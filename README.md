## Installing

Clone repository
```
git clone https://github.com/byabuzyak/fundraise.git
```
Install
``` 
npm install
```
Build
```
npm run build
```
And run the server
```
npm run serve
```

### All done
Now you can visit
```
http://localhost:3000
``` 

## Running the tests
```
npm run test
```
